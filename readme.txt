Clean & simple statistics for your wordpress site.

== Description ==

One click install to have an insight about your wordpress site's visitors.

This easy to use plugin gives you a quick glance at your visitors's time of visits, browser, operating system and much more.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-power-stats.zip` to the `/wp-content/plugins/` directory
2. Unzip `wp-power-stats.zip` in the folder
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= The plug does not work, what should I do? =

Disable and then re-enable the plugin in the 'Plugins' menu in WordPress.
